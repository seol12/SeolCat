  ### 개요
  
  친구가 서비스하는 봇은 이용률이 높고 왜 내 서비스는 이용률이 낮은 걸까,   
  친구의 서비스처럼 이용률이 높으려면 어떻게 해야 될까 하고 문득 생각이 들었습니다.   
  
  생각에 그치지 않고 실행에 옮기기 위해 먼저 서비스를 이용 중인 유저들과,    
  이용하다가 이탈한 유저들에게 의견을 들었습니다.   
  유저들의 의견을 종합해 보니 크게 디자인, 기능 부재, 서비스 속도를 지적했습니다.  
  
  의견들을 참고하여 계획을 세우고, 버전 업데이트를 진행했습니다.   
  
  ```
  1️. 직관적인 인터페이스와 친숙함이 느껴지게 디자인을 적용할 것.   
  
  2. 서비스 이용 시 불편함을 유발했던 부분들을 보완하고 부족했던 기능들을 추가할 것.   
  
  3. 초기 로딩을 개선하여 서비스 이탈률을 감소시킬 것.   
  ```
